HANDS-ON — 40 min Rules Workshop

Write Your Rules

Part 1 — Personal Rules (10 min)

  • 6–8 rules in Cursor Settings → User Rules
  • Error handling, dependencies, AI behavior
  • Test: would you send it back if violated?

Part 2 — Project Rules (15 min)

  • Use Plan mode to draft them
  • Assertions, not requests
  • Save to .cursor/rules/project-rules.mdc

Part 3 — Peer Review (5 min)

Swap rules with a neighbor. Find one to cut and one that's missing.
