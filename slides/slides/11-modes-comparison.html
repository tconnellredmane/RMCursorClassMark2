What Mode When?

ModeBest forAnalogy
Plan Architecture, briefing, design decisions — before any code Briefing a contractor before work starts
Agent Implementation — after the plan is solid; one checkpoint at a time Giving a colleague a task and reviewing their work
Ask Understanding code, reviewing AI output — no changes possible Reading a PR without clicking Approve
Inline Edit Quick, targeted changes to selected code Pointing at a line and saying "change this"

Rule of thumb: Designing? → Plan.   Building? → Agent.   Reviewing? → Ask.
