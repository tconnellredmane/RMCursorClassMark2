The Architect's Five Verbs

Cursor does the doing. You do the deciding.

VerbWhat it means
DefinesWhat to build and what success looks like — before a line is written
DirectsHow the agent should proceed, step by step, checkpoint by checkpoint
DecidesWhich approach, which trade-offs, which alternatives to reject
EvaluatesWhat came back — critically, against the brief, not just "does it run"
ApprovesBefore anything ships — the human gate that closes the loop

If you're not doing these five things, you're a passenger.
